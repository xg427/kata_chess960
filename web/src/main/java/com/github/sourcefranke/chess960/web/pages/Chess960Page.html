Chess960 Starting Position Generator

Demo

This is my proposal for a new code kata.
Every time a new starting position for a Chess960 game will be generated.

Rules

1. Kings and Rooks

The king has to be placed between both rooks.

2. Bishops

The bishops have to be placed on fields of different colors.

3. Others

All other pieces can be placed without any further restrictions.
